# Parent Testimonial Homepage Copy Versions

Use this document to finalise the two quote versions for each parent testimonial before sending the content to the builder.

## Copy rules

Each parent needs two body-copy versions:

* **Featured quote:** 45–75 words. Used when the card is promoted to the large featured card.
* **Secondary quote:** 18–35 words. Used in the smaller card grid.

Avoid children's names unless explicitly approved. Keep parent names, titles, and descriptors only where approved.
